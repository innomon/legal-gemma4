# Track: QLoRA Layer Implementation

Develop the custom `QLoRALinear` layer in GoMLX with PLE integration.

## Objectives
- Implement the `QLoRALinear` struct.
- Handle bitwise unpacking of weights in the forward pass.
- Integrate LoRA A/B matrices with scaling.
- Implement Per-Layer Embedding (PLE) residual signal.

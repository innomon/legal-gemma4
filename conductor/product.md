# Product Definition: Legal-Gemma4

## Vision
A specialized LLM for Indian Law, capable of passing the All India Bar Examination (AIBE) while running efficiently on consumer hardware (Mac M4/Rpi5).

## Target Audience
- Law students in India.
- Legal professionals needing quick domain-specific analysis.
- Researchers in AI for Legal applications.

## Key Features
- **NF4 Quantization:** 4-bit memory efficiency.
- **QLoRA Finetuning:** Targeted domain adaptation on Indian law.
- **PLE Support:** Full architectural compatibility with Gemma 4.
- **Go-Native:** No dependence on Python or external C++ libraries for the core logic.
